    match downloader.download(&task).await {
        Ok(()) => return,
        Err(e) => {
            last_error = Some(e);
            current_retry += 1;
            if current_retry > max_retries { break; }

## 四、用户体验

### 4.1 API 改进

> [!NOTE]
> 由于 TTHSD 作为动态库发布，Builder 模式可能无法跨语言调用，因此采用直接参数传递方式。

**当前 API**:
```python
# Python 接口
dl_id = dl.start_download(
    urls=["https://example.com/file.zip"],
    save_paths=["./file.zip"],
    thread_count=64,
    chunk_size_mb=10,
    headers={"X-Global": "value"},
    task_headers=[{"X-Task": "value"}]
)

# 运行时配置
dl.set_speed_limit(dl_id, 1024 * 1024)  # 1MB/s
dl.set_proxy(dl_id, "http://proxy:8080")
dl.set_retry_config(dl_id, 5, 2000, 60000)
stats = dl.get_performance_stats(dl_id)
```
